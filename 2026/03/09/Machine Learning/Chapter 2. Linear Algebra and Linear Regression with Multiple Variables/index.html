Linear Algebra

Linear Regression with Multiple Variables

The hypothesis can be described as follow:
$$
h_\theta(x) = \theta_0 + \theta_1 x_1 + \theta_2 x_2 +··· + \theta_n x_n
$$
此时模型参数视为一个 n+1 维的向量，任何一个训练实例 $x^{(i)}$ 也是 n+1 维向量（$x^{(i)}_j$ 是第 i 个训练实例的第 j 个特征），特征矩阵的维度是 $m*(n+1)$ 。因此公式可以简化为 $h_\theta=\theta^TX$ ，其中T表示转置。

Then we come to Gradient Descent with Multiple Variables.
Like what in Single-Regression, We also need a loss function.
$$
J(\theta_1,\theta_2,…,\theta_n)=\frac{1}{2m}\sum_{i=1}^m (h_\theta (x^{(i)})-y^{(i)})^2
$$
The Goal is to find a series of Functions that MINIMIZE the Loss Function. So the Batch Gradient Descent of Multiple Variables Linear Regression will be:
$$
Repeat\space(\theta_j=\theta_j-\alpha\frac{\partial J}{\partial \theta_j})
$$
Q: Why BGD “Batch”?
A: Everytime we renew $\theta$, we need to compute all the m training examples, and after computing, we renew all $\theta_i$ in vector $\theta$.
