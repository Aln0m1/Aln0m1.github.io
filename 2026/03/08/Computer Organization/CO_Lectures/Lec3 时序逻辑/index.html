时序逻辑的特点：

  • 当前输出由当前输入与电路原来状态共同决定
  • 电路的状态与时间顺序有关
  • 由组合逻辑电路和存储电路构成

RS：Reset/Set
S和R不能同时为0；其余三种情况实现了置0、置1、锁存。

钟控D锁存器

如何消除CP的影响？
将钟控RS锁存器输入由R、S双端输入改为单端输入（D），即将其S端改为D输入端，D经过非门接R端（S、R总是互反）![[Pasted image 20250929143336.png]]
把锁存功能全部赋予CP=0的情况。

触发器

D触发器

由两个反相的D锁存器组成。![[Pasted image 20250929150940.png]]

Q:工作原理是？

![[Pasted image 20250930092541.png]]
为什么要把clk和clk的非连接到同一个与门上？
这一设计是为了实现时钟的上升沿触发特性，避免“空翻”问题：

  • 当CLK由低跳变到高（上升沿）的瞬间，由于门电路的延迟，CLK和其反相信号会在极短时间内同时处于有效电平，与门会输出一个窄脉冲，触发后续的基本RS触发器更新状态（将D端的值锁存到Q端）；
  • 这种设计确保D触发器仅在时钟上升沿时响应输入并更新状态，在一个时钟周期内只动作一次，解决了电平触发方式下可能出现的“空翻”（多次无意义翻转）问题，保证了时序逻辑的稳定性。
    D触发器还可以带有使能端、复位功能、置位功能

D锁存器与D触发器的Verilog描述

D锁存器为电位触发。假设为高电位触发：![[Pasted image 20250930094615.png]]
![[Pasted image 20250930094637.png]]
Verilog使用Reg描述锁存器和触发器，Reg可以被综合成触发器、锁存器、电线。
always块中由时钟边沿触发的reg被综合为触发器。
always块的条件语句在未覆盖分支默认输出原值，这一逻辑被综合为锁存器。
如果reg变量起到链接或计算作用，就没有存储功能，这时要使用完整的条件语句保证不出现锁存器。

JK触发器

D锁存器虽然没有约束条件，但功能较少（只有置0、置1功能）
我们在钟控RS锁存器的基础上，增加两条反馈线，Q反馈到R钟控门的输入端，并把R改为K；~Q反馈到S门上，并把S改名为J。![[Pasted image 20250930095355.png]]![[Pasted image 20250930095425.png]]
![[Pasted image 20250930095733.png]]
相比于D触发器，实现了保存和翻转的功能。

Moore型有限状态机

有限状态机是描述状态变化的离散数学模型。包括状态集合、状态转移集合、操作集合。构建有限状态机电路需要次态逻辑电路、状态寄存器、输出逻辑电路。![[Pasted image 20250930101750.png]]

如何设计Moore状态机？

![[Pasted image 20250930102300.png]]
步骤：

  1. 搭建状态机框架
  2. 绘制次态逻辑的状态转移图
  3. 根据状态转移图搭建电路图
  4. 写出输出逻辑的真值表
  5. 根据真值表搭建电路图
