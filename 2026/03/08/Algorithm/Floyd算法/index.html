Floyd算法

Floyd算法简介

Floyd-Warshall 算法是解决所有顶点对之间的最短路径问题的经典算法。核心思想是动态规划。
可以处理有向图或无向图，可以处理带负权边的图，不能处理带有负权回路（负环）的图，因为如果存在负环，路径长度可以无限小，算法无法收敛到一个有限值。
假设图中有 n 个顶点，我们使用一个 n x n 的矩阵 dist 来存储顶点对之间的最短路径长度。

  1. 初始化 dist 矩阵：
    1. dist[i][j] 表示从顶点 i 到顶点 j 的初始距离。
    2. 如果 i 和 j 是同一个顶点，dist[i][j] = 0。
    3. 如果 i 和 j 之间有一条直接的边，dist[i][j] 等于该边的权重。
    4. 如果 i 和 j 之间没有直接的边，dist[i][j] 设为一个无穷大的值（例如 INT_MAX）。
  2. 迭代更新 dist 矩阵：
    我们引入一个中间顶点 k（k 的范围从 0 到 n-1）。对于每一个 k，我们都遍历所有的顶点对 (i, j)，并检查：dist[i][j] > dist[i][k] + dist[k][j]
    如果这个不等式成立，说明从 i 经过 k 再到 j 的路径比当前已知的 i 到 j 的路径更短。因此，我们更新 dist[i][j] 的值：
    dist[i][j] = dist[i][k] + dist[k][j]
    这个过程可以理解为：“对于所有的 i 和 j，当允许经过第 k 个顶点时，i 到 j 的最短路径是多少？”。通过 k 从 0 到 n-1 的循环，我们逐步允许所有顶点作为中间点，最终 dist[i][j] 中存储的就是 i 到 j 的全局最短路径。
  3. 输出结果：
    算法结束后，dist 矩阵就是最终的所有顶点对之间的最短路径长度矩阵。
